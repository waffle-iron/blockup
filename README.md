> Koulu projekti

> Android sovellus jonka idea on auttaa kaikenikäisiä kiusaamista kokevia ihmisi. Tarjoamalla tukea ja yhteisö apua


## For Testing

## First step
```
cd into the client/blockup folder
```

## Dev

```
$ npm install
```
### Test

```
$ gulp test
```
### Run

```
$ ionic serve
```

### Build

```
$ ionic build --release android
```
